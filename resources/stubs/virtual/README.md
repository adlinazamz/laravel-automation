# Virtual Stubs

This directory contains template stubs for **virtual runtime controllers and routes**.
They simulate physical Laravel CRUD structures but execute dynamically using table metadata.

- `ApiController.stub`: Virtual REST API
- `Controller.stub`: Virtual Blade controller
- `Model.stub`: Virtual Eloquent-like abstraction
- `ApiRoute.stub` / `WebRoute.stub`: Route templates
- `Swagger.stub`: For dynamic Swagger/OpenAPI generation
- `SideNav.stub`: For sidebar UI entries

Physical files = only for runtime (e.g., VirtualApiController.php)
